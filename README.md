CheapShark Reader

Роль
- Android Developer (Kotlin)

Тип занятости
- Личный проект

Стек технологий
- Kotlin
- Jetpack Compose
- MVVM (Clean Architecture)
- Koin (Dependency Injection)
- Retrofit (REST API)
- Room (локальная база данных)
- StateFlow
- Navigation Compose
- Coil (загрузка изображений)

Цель проекта
- Разработка Android-приложения для поиска видеоигр и просмотра актуальных предложений (скидок) с возможностью работы в оффлайн-режиме.
- Приложение использует публичное API CheapShark: https://apidocs.cheapshark.com/

Функционал
- Поиск игр по названию
- Просмотр списка игр с актуальными ценами
- Просмотр предложений (deals) по выбранной игре
- Отображение магазинов с логотипами
- Добавление/удаление игр в избранное
- Экран избранных игр
- Кэширование данных в локальной базе (Room)
- Частичная работа без интернета (через локальный кэш)
- Автоматическое обновление UI через StateFlow

Мой вклад
- Проектирование архитектуры (Clean Architecture + MVVM)
- Интеграция REST API (Retrofit)
- Реализация локального кэша (Room)
- Настройка Dependency Injection (Koin)
- Разработка UI на Jetpack Compose
- Реализация навигации между экранами
- Работа с асинхронными потоками (Coroutines, Flow)
- Реализация избранного и синхронизации с БД

Результат
- Реализовано полнофункциональное Android-приложение
- Обеспечена работа с данными как онлайн, так и оффлайн
- UI автоматически обновляется при изменении данных
- Приложение соответствует принципам чистой архитектуры
  
Системные требования
- minSdk: 27
- targetSdk: 36
- Android 8.1 (Oreo) и выше
